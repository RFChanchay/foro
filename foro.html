Semejanzas

Diferencias

XML y JSON presentan formato de texto plano.

JSON se basa en JAVA y XML se basa en SGML

Ambos poseen autodescripción con lo cual es aumenta su legibilidad.

JSON soporta varios tipos de datos como arreglos, cadenas, etc. XML no soporta arreglos, pero admite datos como imágenes, gráficos, tablas, etc.

Poseen estructura jerárquica y sus valores pueden contener una lista de objetos o valores.

JSON soporta solamente codificación UTF-8. XML soporta diferentes tipos de codificación.

XML trabaja con etiquetado.

JSON es mas ligero que XML.
